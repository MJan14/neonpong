# Neon Pong

A simple and fun **Neon Pong** game made in **Wick Editor**.  

## How to Play
- **Player 1:** Use `W` (up) and `S` (down) to move the paddle.  
- **Player 2:** Use the `Up Arrow` and `Down Arrow` keys to move the paddle.  

## Features
- Neon-style graphics  
- Classic Pong gameplay  
- Two-player mode  

## How to Run
1. Open the https://mjan14.github.io/neonpong/ file in your browser.  
2. Start playing immediately!  

## Notes
- Made with **Wick Editor**, fully playable in a browser.  
- Best played on desktop for keyboard controls.
